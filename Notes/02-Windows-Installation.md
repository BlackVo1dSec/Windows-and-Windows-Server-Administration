Learning Objectives

After completing this topic, I should be able to:

- [ ] Explain what an operating system is.
- [ ] Describe the Windows installation process.
- [ ] Create a bootable Windows USB.
- [ ] Configure BIOS/UEFI for installation.
- [ ] Perform a clean installation of Windows.
- [ ] Explain partitions and file systems.
- [ ] Complete the Windows first-time setup (OOBE).
- [ ] Install drivers and updates after installation.

---

Prerequisites

Before installing Windows, I need:

- [ ] A compatible computer
- [ ] A USB flash drive (8 GB or larger)
- [ ] Windows ISO file
- [ ] Bootable USB creation tool
- [ ] Internet connection (optional but recommended)
- [ ] Product key (optional during installation)
